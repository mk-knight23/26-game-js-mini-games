# Design System: 11-js-mini-games

## Visual Philosophy
The design follows a **Clean Modern Hub / Laboratory Dashboard** aesthetic. It uses vibrant accent colors for individual apps against a clean, spacious slate-50 canvas.

## Design Patterns
-   **App Tiles**: Rounded (2rem) buttons with distinct shadow profiles and hover-scaling.
-   **Vibrant Accents**: Rose, Blue, Emerald, Violet, and Amber used as primary identifiers for each app.
-   **Sticky Glass Header**: Persistent navigation with backdrop blur for a premium feel.

## Color Palette
-   **Lab Surface**: Slate-50 (#f8fafc)
-   **Primary (Indigo)**: Indigo-600 (#4f46e5)
-   **App 1 (Rose)**: Rose-500 (#f43f5e)
-   **App 2 (Blue)**: Blue-500 (#3b82f6)
-   **App 3 (Emerald)**: Emerald-500 (#10b981)
-   **App 4 (Violet)**: Violet-500 (#8b5cf6)
-   **App 5 (Amber)**: Amber-500 (#f59e0b)

## Interaction Design
-   **Hub Entry**: 20px Y-axis slide-in for the app grid.
-   **Back to Hub**: Persistent navigation allows users to escape sub-apps easily.
-   **Interactive Micro-apps**: Standardized button and input styling across all mini-apps.
